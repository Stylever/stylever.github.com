Build A Blog

First of all, you must register a github account. Then learn more about https://pages.github.com.

I try to use RequireJS, Jquery, Bootstrap and artTemplate to build my blog.

RequireJS is a JavaScript file and module loader. It will improve the speed and quality of your code.

Jquery is a fast, small, and feature-rich JavaScript library. It is easy to learn and very useful.

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. I use it to build a responsive web design.

ArtTemplate is a javascript template engine. It can push a html document to website page instead of using javascript to add Doms. Through it, the language of my blog can be changed.
